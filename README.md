# Flutter To-Do App

A basic yet functional To-Do application built with Flutter that allows users to manage their daily tasks efficiently.

##  Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Dynamic task list using ListView
* Simple dialog-based UI for input
* Clean and responsive design


## Tech Stack

* Flutter
* Dart

### Prerequisites

* Flutter SDK installed
* Android Studio / VS Code
* Emulator or physical device

##  How It Works

* Tasks are stored in a `List<String>`.
* `setState()` updates UI whenever tasks change.
* `AlertDialog` is used for adding and editing tasks.
* `ListView.builder` dynamically renders all tasks.
* FloatingActionButton opens the add-task dialog.

##  Future Improvements

* Add task completion checkbox
* Save tasks locally using SharedPreferences or SQLite
* Add due dates and reminders
* Improve UI with animations and categories
* Add dark mode support
